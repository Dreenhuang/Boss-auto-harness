# AI学习助手 Web版 使用教程

## 快速上手指南

### 第一步：启动项目

双击 `一键启动.bat` 或在命令行运行：

```bash
cd K:\AI\AIxueixi-haha2\vibePM-web
bun install
bun run dev
```

浏览器自动打开：http://localhost:3003/

### 第二步：浏览应用

1. **闪屏页** - 自动显示2秒后跳转
2. **引导页** - 3页滑动引导，可点击"跳过"
3. **问卷页** - 选择学习偏好，点击"开始学习"
4. **首页** - 主界面，包含搜索、分类、瀑布流

### 第三步：主要功能

#### 首页功能

- **搜索栏** - 点击搜索图标进入搜索页
- **分类标签** - 横向滚动切换不同内容类型
- **瀑布流卡片** - 点击卡片进入详情页

#### 底部导航

- **首页** - 返回主页面
- **知识库** - 查看收藏、路径、笔记
- **我的** - 个人中心，设置、导出等

#### 搜索功能

- 输入关键词实时过滤结果
- 查看搜索历史记录
- 点击热门搜索快速搜索

#### 学习路径

- 在"我的"页面点击"学习路径"
- 查看路径进度
- 点击"继续学习"继续未完成的路径

#### 导出笔记

- 在"我的"页面点击"导出笔记"
- 选择导出内容类型
- 选择导出格式（PDF/Markdown/HTML）
- 配置导出选项后点击"导出"

## 常见问题

### Q: 页面显示不正常？

确保已正确安装依赖：`bun install`

### Q: 如何修改样式？

编辑 `src/assets/main.css` 或各组件的 `<style>` 部分

### Q: 如何添加新页面？

1. 在 `src/views/` 创建新 Vue 文件
2. 在 `src/router/index.js` 添加路由配置
3. 重启开发服务器

## 技术说明

- 使用 Vue 3 Composition API
- 使用 Tailwind CSS 进行样式设计
- 使用 Element Plus 作为 UI 组件库
- 使用 Lucide 图标库替代 emoji
- 移动端优先设计，支持 375px-428px 屏幕
