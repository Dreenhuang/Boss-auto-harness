# 100+部电影案例库

> 本文档收录100+部经典电影案例，按照10种电影类型分类，用于救猫咪编剧助手的Few-Shot提示和教学参考。每个类型至少收录10部电影，包含中文电影和好莱坞经典。

## 案例库使用指南

### 在Few-Shot提示中使用

当需要为用户解释某个类型时，可以引用具体案例：

```
如果你想写一个"推理侦探型"的故事，
参考《搏击俱乐部》的结构——
它用"精神分裂"作为核心谜题，
让观众一直以为泰勒·德顿是真实存在的，
直到最后才揭示真相。
```

### 在生成节拍表时使用

当为用户生成节拍表时，可以引用相似案例：

```
基于你的"失忆特工"概念，
我参考了《记忆碎片》的结构：
开场建立主角的困境，
催化剂是他的失忆状态，
中点是第一次接近真相，
结局是最终的"顿悟"时刻。
```

### 在案例库中查找相似案例

当用户描述故事概念时，查找最相似的案例：

| 用户概念 | 推荐案例 | 原因 |
|----------|----------|------|
| 一个人发现自己是超人 | 《超人》 | 身份发现+责任 |
| 失忆的人寻找真相 | 《记忆碎片》 | 记忆碎片结构 |
| 普通人面对外星人入侵 | 《世界末日》 | 普通人vs大灾难 |
| 两个不般配的人相爱 | 《当哈里遇见莎莉》 | 冤家变爱人的经典模式 |
