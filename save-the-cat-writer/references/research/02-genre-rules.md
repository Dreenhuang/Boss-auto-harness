# 10种电影类型完整规则指南

> 本文档是布莱克·斯奈德（Blake Snyder）《救猫咪》电影编剧理论中10种电影类型的完整参考。包含每种类型的核心规则、必备元素、代表案例，以及如何在创作中应用。

---

## 类型概述

斯奈德将所有电影分为10种基本类型。每种类型都有其"不可违背的规则"，理解这些规则是创作自由的前提。

### 类型速览表

| 类型 | 英文名 | 核心问题 | 情感出口 |
|------|--------|----------|----------|
| 鬼怪屋型 | Monster in the House | 我们能逃脱吗？ | 恐惧与解脱 |
| 金羊毛型 | Golden Fleece | 我们能找到吗？ | 冒险与成长 |
| 如愿以偿型 | Out of the Bottle | 如果我得到了呢？ | 讽刺与警示 |
| 麻烦家伙型 | Dude with a Problem | 我如何生存？ | 悬疑与紧张 |
| 变迁仪式型 | Rites of Passage | 我如何面对失去？ | 悲伤与接受 |
| 伙伴之情型 | Buddy Love | 我如何学会爱？ | 温暖与连接 |
| 推理侦探型 | Whydunit | 谁干的？ | 揭秘与正义 |
| 傻瓜获胜型 | The Fool Triumphant | 傻瓜怎么赢的？ | 荒诞与智慧 |
| 被制度化型 | Institutionalized | 个人vs群体，如何抉择？ | 批判与觉醒 |
| 超级英雄型 | Superhero | 我能战胜邪恶吗？ | 敬畏与希望 |

## 类型一：鬼怪屋型（Monster in the House）

### 核心规则

**必须同时包含三个元素：**
1. **一个怪物**（可以是超自然、生物、或人变成的怪物）
2. **一个房子**（封闭空间，有限的逃脱路线）
3. **一种罪恶**（导致怪物出现的原因，通常是人类的贪欲或错误）

### 故事逻辑

因为某种罪恶，怪物被召唤出来。房子里的人必须面对怪物，在恐惧中做出道德选择。

### 必备元素

1. **倒计时压力**：怪物即将到来，必须在某个时间前逃脱
2. **黑暗的秘密**：房子里隐藏着某个秘密
3. **人类的罪恶**：怪物是对人类罪恶的惩罚
4. **封闭空间**：恐惧来自于无处可逃

### 角色设计

**受害者原型：**
- 拥有罪恶秘密的人
- 逃脱能力有限的人
- 在道德选择中挣扎的人

**怪物原型：**
- 贪欲的象征
- 不可阻挡的力量
- 无法沟通的存在

### 节拍表调整

| 节拍 | 鬼怪屋型特殊处理 |
|------|-----------------|
| 开场画面 | 建立"罪恶"的存在 |
| 催化剂 | 怪物的出现或发现 |
| 游戏时间 | 追逐、躲藏、死亡 |
| 中点 | 第一次逃脱成功（假胜利） |
| 一无所有 | 怪物发现最后的藏身之处 |
| 结局 | 道德选择：谁来承担罪恶？ |

### 创新变体

- **心理恐怖型**：《逃出绝命镇》- 怪物是社会隐藏的种族主义
- **喜剧恐怖型**：《惊声尖笑》系列 - 用幽默解构恐怖类型
- **反转型**：《逃出绝命镇》- 受害者变成了另一种"怪物"

### 常见错误

- 缺少"罪恶"元素，怪物只是随机杀人
- "房子"不够封闭，逃脱太容易
- 怪物太晚出现，前半段无聊
- 道德选择不清晰

## 类型二：金羊毛型（Golden Fleece）

### 核心规则

**一群人出发去获得某个目标，途中每个人都在改变。**

外在目标是表，内在目标是重塑自我。

### 故事逻辑

主角必须与一群人完成一段旅程，在旅途中：
- 每个人展示自己的缺陷
- 每个人经历考验
- 每个人获得成长或改变
- 最终，外在目标和内在转变同时完成

### 必备元素

1. **团队成员**：每人都有独特的缺陷和能力
2. **路途规则**：不能轻易获得目标，需要付出代价
3. **勋章时刻**：每个角色在某个时刻展示成长
4. **目的地**：明确的目标地点或物品

### 角色设计

**主角原型：**
- 被选中或被推举的领导者
- 有缺陷但愿意学习
- 在旅程中学会承担责任

**团队原型：**
- 英雄（能力最强但骄傲）
- 傻瓜（提供喜剧但有智慧）
- 恋人（提供情感线）
- 导师（提供智慧但有盲点）
- 背叛者（提供冲突和紧张）

### 节拍表调整

| 节拍 | 金羊毛型特殊处理 |
|------|-----------------|
| 开场画面 | 展示主角的"旧世界"和缺陷 |
| 催化剂 | 接受使命，组建团队 |
| B故事 | 团队成员之间的关系和冲突 |
| 游戏时间 | 旅途中的各种挑战 |
| 中点 | 团队第一次成功或分裂 |
| 一无所有 | 团队几乎瓦解 |
| 结局 | 目标达成，但真正重要的是团队的改变 |

- **反英雄金羊毛**：《老无所依》- 追寻者变成被追猎者
- **女性金羊毛**：《末路狂花》- 女性的觉醒之旅

### 常见错误

- 团队成员太少或太多
- 旅途目标不清晰
- 角色改变不明显
- 内在转变和外在目标脱节

## 类型三：如愿以偿型（Out of the Bottle）

### 核心规则

**主角得到他想要的，却发现这并不是他真正需要的。**

愿望实现 → 发现代价 → 学会真正的教训

### 故事逻辑

这是一个关于"愿望"的讽刺故事。主角想要某样东西，得到后却发现：
- 愿望带来了意想不到的代价
- 追求愿望的过程中失去了更重要的人
- 或者，愿望本身就是一个错误

### 必备元素

1. **魔法元素**：许愿池、精灵、时光机、超能力
2. **讽刺意味**：得到的和想象的不同
3. **代价揭示**：愿望的真正代价是什么
4. **真正需要**：主角真正需要学的是什么

### 角色设计

**主角原型：**
- 对现状不满的人
- 有明确但肤浅的愿望
- 在故事结束时学会感恩

**精灵/赋予者原型：**
- 规则的解释者
- 可以是神秘存在、科技、或者自己的未来/过去

### 节拍表调整

| 节拍 | 如愿以偿型特殊处理 |
|------|-------------------|
| 开场画面 | 展示主角对现状的不满 |
| 催化剂 | 愿望实现的机会出现 |
| 中点 | 愿望第一次实现（假胜利） |
| 一无所有 | 愿望的代价完全显现 |
| 结局 | 放弃愿望（或学会用它做正确的事） |

### 创新变体

- **集体如愿**：《西虹市首富》- 一个月花光十亿
- **反向如愿**：《偷天情缘》- 被困在同一天
- **多人如愿**：《lover别》- 三个平行的如愿故事

### 常见错误

- 愿望的代价不够严重
- 主角太容易学会教训
- 讽刺不够深刻

## 类型四：麻烦家伙型（Dude with a Problem）

### 核心规则

**一个普通人突然陷入不普通的处境。**

故事逻辑：我只想正常生活，但命运不答应。

### 必备元素

1. **普通人身份**：主角是观众可以认同的"你我他"
2. **极端处境**：完全超出日常生活
3. **能力落差**：普通人 vs 极端处境的反差
4. **求生存**：故事的核心是"如何活下来"

### 角色设计

**主角原型：**
- 正在经历人生危机的人
- 有家庭、工作、社会责任的人
- 被卷入非自愿事件的人

**反派原型：**
- 强大的敌人
- 不公正的制度
- 或者单纯的"天意"

### 节拍表调整

| 节拍 | 麻烦家伙型特殊处理 |
|------|-------------------|
| 开场画面 | 展示主角"正常"的生活 |
| 催化剂 | 被卷入危机 |
| 游戏时间 | 尝试各种方法逃脱 |
| 中点 | 第一次接近逃脱（假胜利） |
| 一无所有 | 危机达到顶峰 |
| 结局 | 证明普通人也可以是英雄 |

## 类型五：变迁仪式型（Rites of Passage）

### 核心规则

**关于"失去天真"和"接受责任"的故事。**

故事逻辑：旧世界破碎 → 经历考验 → 建立新世界

### 必备元素

1. **清晰的阶段划分**：童年/成人必须有明确分界
2. **导师角色**：帮助主角理解成人世界
3. **考验**：必须证明主角准备好了
4. **成年礼**：主角接受责任的标志性时刻

### 角色设计

**主角原型：**
- 即将成年的青少年
- 经历人生重大变化的人
- 必须承担新责任的人

**导师原型：**
- 智慧的年长者
- 已经历过同样转变的人
- 可以是父母、老师、或偶然遇到的智者

### 节拍表调整

| 节拍 | 变迁仪式型特殊处理 |
|------|-------------------|
| 开场画面 | 展示"童年"的世界 |
| 催化剂 | 打破旧世界的事件 |
| B故事 | 导师与主角的关系 |
| 游戏时间 | 面对成年人的挑战 |
| 中点 | 第一次"假装"成人 |
| 一无所有 | 旧世界完全破碎 |
| 结局 | 正式进入成人世界 |

### 创新变体

- **反向变迁**：《本杰明·巴顿奇事》- 逆向成长的奇特旅程
- **集体变迁**：《死亡诗社》- 一群学生的觉醒

### 常见错误

- 转变不够清晰
- 导师角色太弱
- 考验不够深刻

## 类型六：伙伴之情型（Buddy Love）

### 核心规则

**两个不完整的人，通过彼此学会成长。**

故事逻辑：相遇 → 冲突 → 并肩作战 → 彼此改变

### 必备元素

1. **互补的缺陷**：两人都有问题，但问题正好互补
2. **势均力敌的张力**：两人应该是对等的
3. **关系的演变**：从讨厌到接受到爱
4. **象征物/任务**：共同的"使命"让两人走到一起

### 角色设计

**伙伴A原型：**
- 强势但冷漠
- 有能力但缺乏情感
- 被动接受而非主动追求

**伙伴B原型：**
- 弱势但热情
- 缺乏能力但有爱心
- 主动追求但被拒绝

### 节拍表调整

| 节拍 | 伙伴之情型特殊处理 |
|------|-------------------|
| 开场画面 | 展示两人各自的生活（平行剪辑） |
| 催化剂 | 两人被迫相遇 |
| B故事 | 两人关系的发展（这是真正的故事） |
| 游戏时间 | 两人的冲突和合作 |
| 中点 | 第一次亲密时刻（假胜利） |
| 一无所有 | 关系破裂或即将分离 |
| 结局 | 两人和解，共同面对 |

| 《伴娘》 | 2011 | 闺蜜团 | 女性情谊 |
| 《疯狂的石头》 | 2006 | 笨贼三人组 | 笨贼对决 |
| 《心花路放》 | 2014 | 失婚男人与流浪歌手 | 治愈之旅 |
| 《追凶者也》 | 2016 | 小人物追凶 | 黑色幽默 |

### 创新变体

- **老年伙伴**：《遗愿清单》- 两个老人的友谊
- **反英雄伙伴**：《落水姻缘》- 争吵不休的冤家
- **敌对伙伴**：《自杀小队》- 坏人与坏人合作

### 常见错误

- 一方太强/太弱，关系不对等
- 没有真正的冲突和张力
- 关系发展太快或太慢

## 类型七：推理侦探型（Whydunit）

### 核心规则

**观众和侦探一起寻找真相。**

故事逻辑：犯罪发生 → 调查 → 发现真相 → 审判

### 必备元素

1. **聪明的谜题**：线索必须公平但隐蔽
2. **可信的线索**：让观众觉得自己也能发现
3. **令人惊讶的真相**：真相出乎意料但合乎逻辑
4. **侦探主角**：观众跟随的调查者

### 角色设计

**侦探原型：**
- 专业调查者（警察、侦探、记者）
- 普通人被迫调查（邻居、受害者家属）
- 有特殊能力的观察者

**嫌疑人原型：**
- 每个人都有动机和机会
- 表面证据指向的人往往是误导
- 真相揭示时的"顿悟"时刻

### 节拍表调整

| 节拍 | 推理侦探型特殊处理 |
|------|-------------------|
| 开场画面 | 犯罪发生或被发现 |
| 催化剂 | 侦探接手案件 |
| B故事 | 侦探的私人生活/情感线 |
| 游戏时间 | 调查过程、线索收集 |
| 中点 | 发现关键线索（假线索） |
| 一无所有 | 调查走入死胡同或误导 |
| 结局 | 真相揭示 + 惩罚/正义 |

### 经典案例（12部）

| 电影 | 年份 | 侦探类型 | 真相类型 |
|------|------|----------|----------|
| 《唐人街》 | 1974 | 私家侦探 | 水权阴谋 |
| 《搏击俱乐部》 | 1999 | 普通人 | 精神分裂 |
| 《非常嫌疑犯》 | 1995 | 警察 | 魔王传说 |
| 《七宗罪》 | 1995 | 警察 | 凶手就是自己 |
| 《记忆碎片》 | 2000 | 失忆者 | 自我欺骗 |
| 《消失的爱人》 | 2014 | 妻子 | 操纵与反操纵 |
| 《利刃出鞘》 | 2019 | 侦探 | 家族阴谋 |
| 《看不见的客人》 | 2016 | 律师 | 双重反转 |
| 《心迷宫》 | 2014 | 村民 | 多重误会 |
| 《风声》 | 2009 | 特工 | 内部叛徒 |
| 《暗杀》 | 2015 | 刺客 | 阴谋 |
| 《秘密访客》 | 2021 | 家长 | 家庭秘密 |

### 创新变体

- **心理悬疑**：《禁闭岛》- 侦探本身就是谜题
- **多人视角**：《罗生门》- 同一事件的不同版本
- **集体侦探**：《无人生还》- 所有人都是嫌疑人

### 常见错误

- 线索不公平（观众无法推理）
- 真相太简单或太复杂
- 侦探太"神"，没有失误
- 惩罚/正义不到位

## 类型八：傻瓜获胜型（The Fool Triumphant）

### 核心规则

**聪明人解决问题的方法，傻瓜完全没想到。**

故事逻辑：所有人都低估了傻瓜，傻瓜用"愚蠢"的方法获胜

### 必备元素

1. **傻子人设**：主角被所有人看不起
2. **众人的嘲笑**：聪明人的傲慢
3. **意想不到的解决方案**：傻瓜用简单方法解决复杂问题
4. **讽刺性**：对"聪明"智慧的讽刺

### 角色设计

**傻瓜原型：**
- 被社会低估的人
- 有"愚蠢"外表但内在智慧
- 坚持自己的方式，不被理解

**聪明人原型：**
- 看不起傻瓜
- 方法太复杂
- 最终被傻瓜打败

### 节拍表调整

| 节拍 | 傻瓜获胜型特殊处理 |
|------|-------------------|
| 开场画面 | 展示傻瓜被嘲笑 |
| 催化剂 | 傻瓜被给予机会 |
| B故事 | 傻瓜的内心世界 |
| 游戏时间 | 傻瓜的"愚蠢"方法 |
| 中点 | 傻瓜的方法似乎在奏效 |
| 一无所有 | 傻瓜被所有人否定 |
| 结局 | 傻瓜用最简单的办法获胜 |

### 经典案例（10部）

| 电影 | 年份 | 傻瓜类型 | 讽刺对象 |
|------|------|----------|----------|
| 《阿甘正传》 | 1994 | 低智商但真诚 | 美国历史 |
| 《雨人》 | 1988 | 自闭天才 | 商业理性 |
| 《我的舅舅》 | 1958 | 笨拙但善良 | 现代社会 |
| 《阿呆与阿瓜》 | 1994 | 愚蠢搭档 | 聪明人的失败 |
| 《大话王》 | 1997 | 说真话的孩子 | 虚伪 |
| 《大内密探零零发》 | 1996 | 发明家 | 武林高手 |
| 《功夫》 | 2004 | 傻子练武 | 武林 |
| 《疯狂的石头》 | 2006 | 笨贼 | 专业盗贼 |
| 《夏洛特烦恼》 | 2015 | 重生者 | 成功学 |
| 《西虹市首富》 | 2018 | 傻钱 | 金钱至上 |

### 创新变体

- **傻子vs傻子**：《阿呆与阿瓜》- 两个傻瓜的冒险
- **反英雄傻瓜**：《无名之辈》- 小人物的尊严
- **傻子的智慧**：《大话西游》- 至尊宝的爱情傻瓜

### 常见错误

- 主角不够"傻"，太正常
- 傻瓜的胜利太容易
- 没有真正的讽刺性

## 类型九：被制度化型（Institutionalized）

### 核心规则

**个人对抗群体。**

故事逻辑：加入群体 → 被同化 → 觉醒 → 逃离/改革

### 必备元素

1. **强大的群体**：有明确的规则和价值观
2. **群体的"合理性"**：群体的逻辑听起来很有道理
3. **个人的代价**：群体对个人的压迫
4. **觉醒与选择**：主角必须做出选择

### 角色设计

**群体原型：**
- 公司
- 军队
- 黑帮
- 邪教
- 家庭
- 政府机构

**个人原型：**
- 新加入者
- 质疑者
- 领导者

### 节拍表调整

| 节拍 | 被制度化型特殊处理 |
|------|-------------------|
| 开场画面 | 展示群体世界 |
| 催化剂 | 个人与群体的第一次冲突 |
| B故事 | 个人的私人生活/家庭 |
| 游戏时间 | 个人被同化/质疑 |
| 中点 | 个人开始反抗 |
| 一无所有 | 群体对个人的惩罚 |
| 结局 | 逃离/改革/或被完全同化 |

### 经典案例（10部）

| 电影 | 年份 | 制度类型 | 选择类型 |
|------|------|----------|----------|
| 《飞越疯人院》 | 1975 | 精神病院 | 逃离 |
| 《教父》 | 1972 | 黑帮家族 | 被同化 |
| 《黑帮大哥不出庭》 | 1993 | 黑帮 | 忠诚 |
| 《美国黑帮》 | 2007 | 毒贩帝国 | 崛起与陨落 |
| 《毒品网络》 | 2000 | 贩毒系统 | 同流合污 |
| 《华尔街》 | 1987 | 金融 | 贪婪 |
| 《华尔街之狼》 | 2013 | 华尔街 | 疯狂 |
| 《大而不倒》 | 2011 | 金融系统 | 危机 |
| 《大空头》 | 2015 | 金融系统 | 先知 |
| 《监守自盗》 | 2010 | 全球金融 | 贪婪 |

### 创新变体

- **喜剧制度化**：《宿醉》- 单身派对作为"制度"
- **家庭制度化**：《饮食男女》- 传统家庭的控制
- **反英雄制度化**：《黑吃黑》- 罪犯的道德

### 常见错误

- 群体太简单，没有"合理性"
- 个人的选择没有代价
- 结局太明确（完全逃离vs完全同化）

## 类型十：超级英雄型（Superhero）

### 核心规则

**拥有特殊能力的人，面对更强大的敌人。**

故事逻辑：能力与责任、孤独与归属、人性 vs 神性

### 必备元素

1. **超能力设定**：明确的能力范围和限制
2. **标志性的反派**：与主角对等的威胁
3. **道德困境**：能力越大，责任越大
4. **双重身份**：凡人与英雄的张力

### 角色设计

**英雄原型：**
- 被选中的普通人
- 拥有古老力量
- 被实验/变异
- 神/外星人

**反派原型：**
- 镜子化的反派（与英雄相似但选择不同）
- 纯粹的邪恶
- 系统性的威胁

### 节拍表调整

| 节拍 | 超级英雄型特殊处理 |
|------|-------------------|
| 开场画面 | 展示英雄的"凡人身" |
| 催化剂 | 超能力被发现/召唤 |
| B故事 | 英雄的凡人之爱/友情 |
| 游戏时间 | 使用超能力战斗 |
| 中点 | 英雄似乎占上风（假胜利） |
| 一无所有 | 反派展示真正力量 |
| 结局 | 人性战胜/接受神性 |

| 电影 | 年份 | 英雄类型 | 核心冲突 |
|------|------|----------|----------|

| 《蝙蝠侠》 | 2005 | 无超能力 | 人性vs复仇 |
| 《X战警》 | 2000 | 变种人 | 歧视 |

| 《雷神》 | 2011 | 神 | 傲慢 |
| 《绿巨人》 | 2008 | 变异 | 控制 |
| 《奇异博士》 | 2016 | 魔法师 | 傲慢 |

### 创新变体

- **反英雄**：《死侍》- 打破第四墙的超级英雄
- **普通英雄**：《海扁王》- 普通少年变英雄
- **现实英雄**：《全民超人汉考克》- 超能力者的尴尬生活

### 常见错误

- 超能力太无敌，缺少挑战
- 反派太弱
- 双重身份张力不够
- 道德困境不深刻

## 类型混合指南

### 常见的成功混合

| 混合类型 | 代表案例 | 效果 |
|----------|----------|------|
| 推理+鬼怪 | 《小岛惊魂》 | 心理恐惧+悬疑 |
| 金羊毛+伙伴 | 《银河护卫队》 | 冒险+团队情感 |
| 傻瓜+推理 | 《唐人街》 | 讽刺+悬疑 |
| 超级+伙伴 | 《复仇者联盟》 | 能力+团队 |
| 变迁+如愿 | 《大话西游》 | 成长+爱情 |
| 麻烦+推理 | 《全民目击》 | 悬疑+情感 |

### 混合规则

1. **选择一个主类型**：保证核心结构清晰
2. **添加辅助类型元素**：丰富故事但不混乱
3. **保持类型承诺**：观众对类型的期望必须满足
4. **创新要有逻辑**：混合必须是"有机"的

### 混合检查清单

- [ ] 我的故事以哪种类型为主？
- [ ] 我混合了哪几种类型？
- [ ] 每种类型的核心规则是否都得到尊重？
- [ ] 观众能否识别我的故事类型？
- [ ] 我的创新是否"有机"而非强行混合？

## 类型选择指南

### 选择类型的决策树

```
你的故事关于什么？
│
├─ 关于恐惧和逃脱 → 鬼怪屋型
│
├─ 关于旅程和改变 → 金羊毛型
│
├─ 关于愿望和代价 → 如愿以偿型
│
├─ 关于生存和普通人 → 麻烦家伙型
│
├─ 关于成长和失去 → 变迁仪式型
│
├─ 关于爱情和伙伴 → 伙伴之情型
│
├─ 关于真相和调查 → 推理侦探型
│
├─ 关于讽刺和智慧 → 傻瓜获胜型
│
├─ 关于群体和个人 → 被制度化型
│
└─ 关于能力和责任 → 超级英雄型
```

### 类型与情感对应

| 情感体验 | 推荐类型 |
|----------|----------|
| 恐惧与紧张 | 鬼怪屋、推理侦探 |
| 温暖与治愈 | 伙伴之情、变迁仪式 |
| 冒险与刺激 | 金羊毛、超级英雄 |
| 讽刺与幽默 | 傻瓜获胜、如愿以偿 |
| 悬疑与惊奇 | 推理侦探、麻烦家伙 |
| 社会批判 | 被制度化型 |
