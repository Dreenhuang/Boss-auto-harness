### 1.7 动效规范

#### 时长规范

| 动效名称 | 时长 | 使用场景 |
|---------|------|---------|
| Instant | 0ms | 无过渡 |
| Fast | 150ms | 微交互、按钮状态变化 |
| Normal | 250ms | 页面元素出现、消失 |
| Slow | 350ms | 页面切换、弹窗出现 |
| Slower | 500ms | 大型弹窗、重型动效 |

#### 缓动曲线

| 缓动名称 | 数值 | 使用场景 |
|---------|------|---------|
| Ease Out | cubic-bezier(0, 0, 0.2, 1) | 元素出现 |
| Ease In | cubic-bezier(0.4, 0, 1, 1) | 元素消失 |
| Ease In Out | cubic-bezier(0.4, 0, 0.2, 1) | 页面切换 |
| Spring | cubic-bezier(0.175, 0.885, 0.32, 1.275) | 弹跳效果 |

#### 页面切换动效

| 切换场景 | 动效描述 |
|---------|---------|
| 首页 → 卡片详情 | 卡片详情页从底部向上滑入（350ms），背景渐暗 |
| 卡片详情 → 返回 | 下滑超过30%触发关闭，或点击返回按钮 |
| 底部Tab切换 | 淡入淡出（200ms） |
| 标签栏切换 | 标签内容淡入（150ms） |

#### 卡片交互动效

| 交互 | 动效描述 |
|------|---------|
| 卡片悬停 | 轻微上浮2px + 阴影加深（150ms） |
| 收藏点击 | 星星实心 + 轻微弹跳（200ms spring） |
| 左滑卡片 | 平滑左滑，显示"不感兴趣"按钮 |
| 下拉刷新 | 刷新动画旋转（持续） |

## 四、交互动效规格

### 4.1 页面切换

| 切换场景 | 时长 | 缓动曲线 | 描述 |
|---------|------|---------|------|
| Tab切换 | 200ms | ease-out | 内容淡入 |
| 页面滑入（卡片详情） | 350ms | ease-out | 从底部向上 |
| 页面滑出 | 250ms | ease-in | 向底部滑出 |
| 弹窗出现 | 250ms | ease-out | 从底部向上，遮罩淡入 |
| 弹窗关闭 | 200ms | ease-in | 向底部滑出，遮罩淡出 |

### 4.2 卡片交互

| 交互 | 时长 | 缓动曲线 | 描述 |
|------|------|---------|------|
| 卡片悬停 | 150ms | ease-out | 上浮2px，阴影加深 |
| 收藏点击 | 200ms | spring | 星星弹跳，实心变色 |
| 左滑卡片 | 200ms | ease-out | 滑动+显示删除按钮 |
| 下拉刷新 | 持续 | linear | 旋转动画 |
| 加载更多 | 150ms | ease-out | 新卡片淡入 |

### 4.3 按钮状态

| 状态切换 | 时长 |
|---------|------|
| Default → Hover | 150ms |
| Hover → Active | 100ms |
| Active → Disabled | 150ms |
| Loading态 | 持续旋转，1s/圈 |

## 五、标注与切图规范

### 5.1 设计稿标注

设计师交付Figma稿时需包含：
- 每个元素的 x/y 坐标
- 宽/高尺寸
- 背景色/文字色/圆角值
- 字体名称/字号/字重
- 间距标注（标注关键间距，如同模块内间距、模块间间距）

### 5.2 切图资源

| 类型 | 格式 | 尺寸 | 用途 |
|------|------|------|------|
| App图标 | PNG | 1024x1024 | 应用商店上架 |
| TabBar图标 | SVG | 24x24 | 底部导航 |
| 功能图标 | SVG | 20x20 / 24x24 / 32x32 | 各场景使用 |
| 引导页插图 | PNG @2x/@3x | 按设计稿 | 首次引导 |
| 空状态插图 | PNG @2x/@3x | 120x120 | 空状态显示 |
| 二维码 | PNG | 200x200 | 微信导流 |

### 5.3 多端适配

| 设备 | 屏幕宽度 | 备注 |
|------|---------|------|
| iPhone SE | 375px | 最小适配 |
| iPhone 标准 | 390px | 主流 |
| iPhone Pro Max | 430px | 大屏 |
| 安卓主流 | 360-420px | 范围覆盖 |
| iPad | 768px+ | 预留 |

瀑布流卡片列宽自动计算：
```
列宽 = (屏幕宽度 - 16px边距 - 8px列间距) / 2
```
