## 🚀 安装步骤

### 方法一：使用安装脚本（推荐）⭐⭐⭐

1. **打开 PowerShell（管理员）**

2. **运行安装脚本**
   ```powershell
   & "K:\chanbpin4.0\120.产品经理技能包 4.0\trae-agents\setup-keyword-activation.ps1"
   ```

3. **等待安装完成**
   - 自动创建目录
   - 复制AGENT文件
   - 生成规则配置
   - 显示激活关键字

4. **重启 TRAE IDE**

### 方法二：手动安装

1. **复制AGENT文件**
   - 源：`K:\chanbpin4.0\120.产品经理技能包 4.0\trae-agents\boss-product-manager.md`
   - 目标：`C:\Users\Administrator\.trae\agents\`

2. **创建规则目录**
   - 路径：`C:\Users\Administrator\.trae\rules\`

3. **复制规则文件**
   - 源：`K:\chanbpin4.0\120.产品经理技能包 4.0\trae-agents\.boss-rules`
   - 目标：`C:\Users\Administrator\.trae\rules\boss.rules`

4. **重启 TRAE IDE**

## 🔧 配置说明

### 关键字配置文件

**位置**：`~/.trae/rules/boss.rules`

**内容结构**：
```markdown
# boss Product Manager - Keyword Activation Rules

## Keyword Triggers
- "boss" → Activate main agent
- "Boss-harness" → Activate main agent
- "Boss-harness" → Activate main agent
- "boss模式" → Activate Boss Mode
- "Boss-harness" → Activate v4.0 workflow
- "chanpin4.0" → Activate v4.0 workflow

## Shortcut Commands
- "/boss" → Quick activation
- "/boss" → Boss mode
- "/pm" → Product manager mode
```

### 主AGENT文件

**位置**：`~/.trae/agents/boss-product-manager.md`

**关键字定义**：
```yaml
keywords:
  - boss
  - Boss-harness
  - Boss-harness
  - boss模式
  - boss mode
  - Boss-harness
  - chanpin4.0
  - /boss
  - /boss
  - /pm
```

## ⚙️ 自定义配置

### 添加新的关键字

编辑 `~/.trae/agents/boss-product-manager.md` 文件：

```yaml
---
keywords:
  - boss
  - Boss-harness
  - Boss-harness
  - 新关键字1  # 添加新关键字
  - 新关键字2  # 添加新关键字
---
```

### 修改激活模式

编辑 `~/.trae/rules/boss.rules` 文件：

```markdown
## Keyword Triggers
- "你的关键字" → 激活指定模式
```

## 🆘 故障排除

### Q1：关键字不生效？

**解决方案**：
1. 确认已安装 AGENT 文件：
   ```powershell
   Test-Path "$env:USERPROFILE\.trae\agents\boss-product-manager.md"
   ```
2. 重启 TRAE IDE
3. 检查关键字拼写（区分大小写）

### Q2：如何验证安装成功？

**解决方案**：
运行安装脚本，查看输出：
```
SUCCESS! Setup completed!
Found XX AGENT files
```

### Q3：如何卸载关键字激活？

**解决方案**：
```powershell
# 删除AGENT文件
Remove-Item "$env:USERPROFILE\.trae\agents\boss-product-manager.md"

# 删除规则文件
Remove-Item "$env:USERPROFILE\.trae\rules\boss.rules"
```
